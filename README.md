## NO TERMUX

```bash
apt update && apt upgrade
apt install git -y
apt install nodejs -y
apt install ffmpeg -y
git clone https://github.com/Dr3z1n/estelar-bot
cd Estelar-bot
npm install
```

## EM CASO DE ERRO, INSTALE O MÓDULO PARA TERMUX

```bash
pkg install yarn
yarn
```

## Instalando
```bash
$ node .
```
